설치 방법 (최초 1회)

1단계: Tampermonkey 설치

  1. 크롬 열기
  2. 크롬 웹스토어에서 "Tampermonkey" 검색 → 설치
  3. 설치 확인: 크롬 우측 상단에 검은 사각형 아이콘 생김

2단계: 매크로 스크립트 설치

  1. 크롬 주소창에 아래 주소 입력 → Enter:
https://raw.githubusercontent.com/pakjilly-cpu/plazacc-macro/master/plazacc-tampermonkey-v2.user.js
  1. Tampermonkey 설치 화면이 뜨면 → "설치" 클릭
  2. 끝!
설치 끝! 이제 플라자CC 사이트에 들어가면 매크로가 자동으로 뜹니다.

사용법 (매주 월요일)

  1. 크롬에서 플라자CC 사이트 접속 → 로그인
  2. 용인 선택 → 회원번호 "확인" 클릭
  3. 예약할 날짜 클릭 (시간표 나옴)
  4. 오른쪽에 초록색 매크로 패널이 자동으로 뜸
  5. 시간/코스 확인 후 "10시 대기" 클릭
  6. 10시 되면 자동으로 예약 클릭됨!
  7. 팝업이 뜨면 → 비밀번호, 보안문자 입력 (천천히 해도 됨)
9시 55분쯤 미리 들어와서 6번까지 해놓고 기다리세요!

문제 해결

매크로 패널이 안 뜨는 경우

  • 페이지 새로고침(F5) 후 날짜 다시 클릭
  • Tampermonkey 아이콘 클릭 → 스크립트가 활성화되어 있는지 확인

"조건에 맞는 슬롯 없음" 나오는 경우

  • 해당 시간대가 이미 다 예약됨 → 시간 범위를 넓혀보세요
  • 날짜를 제대로 선택했는지 확인

스크립트 업데이트 방법

  1. Tampermonkey 아이콘 → 대시보드 → 기존 스크립트 삭제
  2. 2단계 다시 진행 (최신 버전 자동 설치)
