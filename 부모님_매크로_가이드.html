플라자CC 골프 예약 매크로 사용 가이드 (부모님용)

이 매크로를 사용하면 월요일 10시 예약 오픈이나
취소표를 자동으로 잡아줍니다.

컴퓨터 앞에서 눈을 부릅뜨고 기다릴 필요 없이,
버튼 한 번이면 매크로가 알아서 합니다.

1 설치 방법

처음 한 번만 하면 됩니다. 약 2분 소요.

1
크롬(Chrome) 브라우저를 엽니다
인터넷 아이콘 중에서 동그란 빨강/노랑/초록/파랑 아이콘입니다.
반드시 크롬이어야 합니다. 엣지나 네이버 웨일은 안 됩니다.
2
주소창에 아래 주소를 입력하고 엔터
chrome://extensions
화면 맨 위에 있는 긴 주소창(네이버 검색하는 곳)에 그대로 적으시면 됩니다.
3
오른쪽 위에 "개발자 모드" 스위치를 켭니다
개발자 모드
회색이면 꺼진 것, 파란색(또는 초록색)이면 켜진 것입니다.
4
"압축해제된 확장 프로그램을 로드합니다" 클릭
화면 왼쪽 위에 버튼 3개가 나타납니다. 그중 가장 왼쪽 버튼을 누르세요.
5
폴더 선택 창이 뜨면 아래 폴더를 찾아 선택
chrome-extension 폴더
승환이가 바탕화면에 바로가기를 만들어 드릴 겁니다.
"폴더 선택" 버튼을 누르면 끝!
6
설치 완료!
확장 프로그램 목록에 "플라자CC 예약 매크로"가 보이면 성공입니다.
이제 플라자CC 사이트에 접속하면 매크로가 자동으로 나타납니다.

2 매크로 화면 설명

플라자CC 예약 페이지에 들어가면 화면 오른쪽 위에 아래와 같은 작은 창이 뜹니다.

플라자CC 매크로 v12
09:55:32
목표 날짜
13,14,15
10시 달력 자동 새로고침
시간 범위
10시 ~ 14시
코스 우선순위 (고정): 타이거OUT → IN → 라이온OUT → IN
스캔
10시자동
취소감시
설정 후 버튼을 누르세요.

각 부분 설명

시계 - 현재 시각이 실시간으로 표시됩니다. 10시 정각 확인할 때 유용합니다.
목표 날짜 - 예약하고 싶은 날짜의 "일(日)"만 입력합니다.
예: 4월 13일, 14일, 15일이면 → 13,14,15
여러 날짜를 쉼표(,)로 구분합니다. 앞에서부터 우선 시도합니다.
시간 범위 - 원하는 시간대를 설정합니다.
예: 10시~14시로 하면 오전 10시부터 오후 2시 사이 시간만 잡아줍니다.
코스 우선순위 (자동 고정)
같은 시간에 여러 코스가 비어있으면 아래 순서로 잡아줍니다:
순위코스설명
1순위타이거 OUT가장 먼저 시도
2순위타이거 IN
3순위라이온 OUT
4순위라이온 IN

3 버튼 설명

스캔
확인만 합니다 (예약 안 함)
지금 화면에 보이는 시간표에서 조건에 맞는 빈 자리가 있는지 확인합니다.
테스트할 때 사용하세요.
10시자동
월요일 10시 오픈용
10시 정각에 자동으로 새로고침하고, 빈 자리를 찾아 예약 버튼을 눌러줍니다.
한 번만 시도합니다.
취소감시
취소표 잡기용
이미 마감된 날짜에서 누군가 취소하면 바로 잡아줍니다.
3초마다 계속 확인합니다. 빈 자리 나올 때까지 무한 반복!
중지
작업을 멈춥니다
10시자동이나 취소감시가 실행 중일 때 나타납니다.
누르면 모든 자동 작업이 즉시 멈춥니다.

4 10시 자동예약 사용법

매주 월요일 10시에 그 다음주 예약이 열립니다.
매크로가 10시 정각에 자동으로 빈 자리를 잡아줍니다!
내가 할 일 1
월요일 9시 55분에 플라자CC 사이트 로그인
↓
내가 할 일 2
목표 날짜와 시간 범위 설정
↓
내가 할 일 3
10시자동 버튼 클릭!
↓
이 아래는 매크로가 자동으로 합니다
10시 정각 → 달력 자동 새로고침
↓
목표 날짜 자동 클릭
↓
빈 자리 스캔 (시간 + 코스 조건)
↓
조건에 맞는 자리 발견!
예약 버튼 자동 클릭 + 알림음
↓
팝업 창이 뜹니다!
여기서 보안문자 입력하고 "확인" 누르면 끝!
중요! 보안문자(그림 속 숫자)를 입력하는 것은 직접 하셔야 합니다.
팝업 창이 뜨면 그림에 보이는 숫자를 입력하고 "확인"을 누르세요.

상세 순서

1
월요일 9시 55분에 플라자CC 예약 사이트에 로그인합니다.
2
예약 화면(시간표가 보이는 화면)까지 들어갑니다.
오른쪽 위에 매크로 창이 보이는지 확인합니다.
3
목표 날짜에 예약하려는 날짜를 입력합니다.
예: 다음주 토요일이 15일이면 → 15
토, 일 둘 다 괜찮으면 → 15,16
4
시간 범위를 설정합니다.
보통 10시~14시가 적당합니다.
5
"10시자동" 버튼을 누릅니다.
화면에 "10시 자동예약 대기 중"이라고 나옵니다.
6
가만히 기다립니다. 10시가 되면 자동으로 시작됩니다.
다른 탭이나 다른 프로그램으로 가지 마세요!
7
삐삐삐 소리가 나고 팝업이 뜨면
보안문자 입력 → "확인" 클릭 → 예약 완료!

5 취소표 감시 사용법

이미 마감된 날짜라도 누군가 예약을 취소하면
매크로가 3초마다 확인해서 바로 잡아줍니다!
집안일 하다가 오셔도 됩니다.
내가 할 일 1
플라자CC 사이트 로그인 + 예약 화면 진입
↓
내가 할 일 2
목표 날짜와 시간 범위 설정
↓
내가 할 일 3
취소감시 버튼 클릭!
↓
매크로가 알아서 합니다 (무한 반복)
목표 날짜 시간표 확인
↓
빈 자리 없으면?
3초 기다렸다가 다시 확인 (무한 반복)
↓
빈 자리 발견!
예약 버튼 자동 클릭 + 알림음
↓
팝업 창이 뜹니다!
보안문자 입력 → "확인" 누르면 끝!
편하게 쓰는 법:
취소감시를 켜놓고 집안일을 하세요.
컴퓨터에서 삐삐삐 소리가 나면 와서 보안문자만 입력하면 됩니다!
(단, 컴퓨터 소리를 켜놓으셔야 합니다)

상세 순서

1
플라자CC 예약 사이트에 로그인하고 예약 화면까지 들어갑니다.
2
목표 날짜에 원하는 날짜를 입력합니다.
예: 토요일 15일, 일요일 16일 → 15,16
3
시간 범위를 설정합니다.
4
"취소감시" 버튼을 누릅니다.
화면에 "취소표 감시 시작"이라고 나옵니다.
5
그냥 놔두세요! 매크로가 3초마다 자동으로 확인합니다.
화면이 저절로 계속 새로고침됩니다. 정상입니다!
6
삐삐삐 소리가 나면 컴퓨터 앞으로 와서
팝업에 보안문자 입력 → "확인" → 예약 완료!

6 주의사항

  1. 10시자동은 9시 55분부터 대기하세요
    다른 탭이나 다른 프로그램으로 가면 안 됩니다!
    매크로가 있는 탭을 계속 보고 있어야 합니다.
  2. 팝업이 뜨면 보안문자만 입력하면 됩니다
    그림에 있는 숫자를 입력하고 "확인"을 누르세요.
    이것만 직접 하시면 나머지는 매크로가 다 합니다.
  3. 취소감시는 컴퓨터 소리를 켜두세요
    빈 자리를 찾으면 삐삐삐 소리가 납니다.
    소리를 꺼두면 빈 자리를 찾아도 모를 수 있습니다.
  4. "중지" 버튼을 실수로 누르지 마세요
    매크로가 실행 중일 때 "중지"를 누르면 자동 작업이 멈춥니다.
    다시 시작하려면 버튼을 다시 눌러야 합니다.
  5. 컴퓨터가 잠자기(절전) 모드에 들어가면 안 됩니다
    취소감시를 오래 켜둘 때는 컴퓨터 절전 모드를 해제하세요.
  6. 인터넷이 끊기면 매크로도 멈춥니다
    와이파이가 잘 연결되어 있는지 확인하세요.
