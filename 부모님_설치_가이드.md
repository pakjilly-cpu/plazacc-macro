# 플라자CC 예약 매크로 - 부모님 PC 설치 가이드

## 1단계: 크롬에 Tampermonkey 설치
1. 크롬 열기
2. 크롬 웹스토어에서 "Tampermonkey" 검색 → 설치
3. 설치 확인: 크롬 우측 상단에 검은 사각형 아이콘 생김

## 2단계: 매크로 스크립트 설치
1. 크롬 주소창에 아래 입력 → Enter:
```
https://raw.githubusercontent.com/pakjilly-cpu/plazacc-macro/master/plazacc-tampermonkey-v2.user.js
```
2. Tampermonkey 설치 화면 뜨면 → "설치" 클릭
3. 끝!

> ⚠️ 만약 Tampermonkey가 안 되면 (사무실처럼), 아래 "수동 방법" 참고

## 사용법 (부모님께 알려드릴 내용)

### 매주 월요일 예약하기
1. 크롬에서 **플라자CC 로그인** (plazacc.co.kr)
2. **용인** 선택 → 회원번호 **확인** 클릭
3. 예약할 **날짜** 클릭 (시간표 나옴)
4. 오른쪽에 **초록색 매크로 패널**이 자동으로 뜸
5. 시간/코스 확인 후 **"10시 대기"** 클릭
6. 10시 되면 자동으로 예약 클릭됨!
7. **팝업**이 뜨면 → 비밀번호, 보안문자 입력 (천천히 해도 됨)

### 매크로 패널 버튼 설명
- **스캔**: 현재 예약 가능한 시간 목록 확인
- **바로 클릭!**: 지금 바로 조건에 맞는 예약 클릭
- **10시 대기**: 10시까지 기다렸다가 자동 클릭 (매주 월요일용)
- **중지**: 대기 중지

### 설정 변경
- **시간 범위**: 원하는 시간대 선택 (기본 06:00~17:00)
- **코스 우선순위**:
  - 타이거OUT 우선 (전체) ← 기본값
  - 타이거IN 우선 (전체)
  - 타이거OUT만
  - 타이거IN만

## 수동 방법 (Tampermonkey 안 될 때)

### 최초 1회 준비
1. 아래 주소에서 스크립트 다운로드:
```
https://raw.githubusercontent.com/pakjilly-cpu/plazacc-macro/master/plazacc-snippet.js
```
2. "다른 이름으로 저장" → 바탕화면에 `매크로.js`로 저장

### 매번 사용할 때
1. 플라자CC 로그인 → 용인 → 확인 → 날짜 클릭
2. **F12** 누르기 (개발자 도구 열림)
3. **Console** 탭 클릭
4. 상단 **"top ▼"** 드롭다운 → **"ifrmStep3"** 선택
5. `allow pasting` 타이핑 → Enter
6. 바탕화면 `매크로.js`를 메모장으로 열기 → Ctrl+A → Ctrl+C
7. Console에 Ctrl+V → Enter
8. 매크로 패널 뜸 → 10시 대기 클릭

> 날짜 바꾸면: Console에서 ↑(위 화살표) → Enter 하면 다시 실행됨

## 문제 해결

### 매크로 패널이 안 뜨는 경우
- 페이지 새로고침(F5) 후 날짜 다시 클릭
- Tampermonkey 아이콘 클릭 → 스크립트가 활성화되어 있는지 확인

### "조건에 맞는 슬롯 없음" 나오는 경우
- 해당 시간대가 이미 다 예약됨 → 시간 범위를 넓혀보세요
- 날짜를 제대로 선택했는지 확인

### 스크립트 업데이트 방법
1. Tampermonkey 아이콘 → 대시보드 → 기존 스크립트 삭제
2. 2단계 다시 진행 (최신 버전 자동 설치)
